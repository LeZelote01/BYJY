# Requirements pour l'Extracteur de Métadonnées
# =========================================

# Dépendances principales pour l'extraction de métadonnées

# Images (EXIF, métadonnées visuelles)
Pillow>=10.0.0                  # Traitement d'images et extraction EXIF

# Documents PDF
PyPDF2>=3.0.1                  # Extraction métadonnées PDF

# Audio et Vidéo (tags, propriétés)
mutagen>=1.47.0                # Tags audio/vidéo (MP3, FLAC, OGG, MP4, etc.)

# Documents Office
python-docx>=0.8.11            # Métadonnées documents Word (.docx)

# Modules Python standard utilisés:
# - os : opérations système et fichiers
# - sys : informations système
# - argparse : interface ligne de commande
# - json : export format JSON
# - csv : export format CSV
# - datetime : timestamps et dates
# - pathlib : manipulation chemins modernes
# - hashlib : calcul d'empreintes (MD5, SHA1, SHA256)
# - mimetypes : détection types MIME
# - tkinter : interface graphique (inclus avec Python)
# - tkinter.ttk : widgets modernes
# - tkinter.filedialog : sélection fichiers
# - tkinter.messagebox : boîtes de dialogue
# - tkinter.scrolledtext : zones de texte avec scrollbars
# - threading : exécution background pour GUI
# - glob : recherche fichiers avec wildcards

# Version Python recommandée: 3.8+

# Installation:
# pip install -r requirements.txt

# Notes d'installation par système:

# Linux (Ubuntu/Debian):
# sudo apt-get install python3-tk          # Pour tkinter si non inclus
# sudo apt-get install libjpeg-dev         # Pour Pillow
# sudo apt-get install zlib1g-dev          # Pour Pillow

# macOS avec Homebrew:
# brew install python-tk                   # Pour tkinter si nécessaire

# Windows:
# Généralement aucune dépendance système supplémentaire

# Dépendances optionnelles (pour fonctionnalités avancées):
# python-magic>=0.4.27          # Détection type fichier avancée (Linux/Mac)
# python-magic-bin>=0.4.14      # Version Windows de python-magic
# exifread>=3.0.0               # Alternative à Pillow pour EXIF
# ffmpeg-python>=0.2.0          # Métadonnées vidéo avancées via FFmpeg
# openpyxl>=3.1.0               # Support fichiers Excel (.xlsx)
# python-pptx>=0.6.21           # Support présentations PowerPoint

# Pour développement et packaging:
# pyinstaller>=5.13.0           # Création exécutables
# cx-Freeze>=6.15.0             # Alternative pour exécutables
# auto-py-to-exe>=2.40.0        # Interface graphique pour pyinstaller

# Formats supportés avec les dépendances installées:
# Images: JPEG, PNG, TIFF, BMP, GIF (via Pillow)
# Audio: MP3, FLAC, OGG, M4A, AAC, WAV (via mutagen)  
# Vidéo: MP4, AVI, MKV, MOV (métadonnées basiques via mutagen)
# Documents: PDF (via PyPDF2), DOCX (via python-docx), TXT
# Génériques: Tous formats (analyse signatures binaires)

# Sécurité:
# Toutes les bibliothèques utilisées sont maintenues et sécurisées
# Pillow: régulièrement mis à jour pour corriger vulnérabilités
# PyPDF2: fork maintenu de PyPDF avec corrections sécurité
# mutagen: bibliothèque stable et éprouvée
